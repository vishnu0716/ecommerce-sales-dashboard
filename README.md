# 📊 E-Commerce Sales Dashboard (Power BI)

## 🚀 Overview
This project presents an interactive Power BI dashboard designed to analyze e-commerce sales performance, customer behavior, and profitability.

## 🔍 Features
- KPI tracking: Sales, Profit, Quantity, AOV
- State-wise performance analysis
- Category & sub-category insights
- Monthly profit trends
- Customer and payment mode analysis

## 📈 Key Insights
- Clothing category dominates sales
- COD is the most used payment method
- Certain months show profit decline
- Top customers contribute significantly to revenue

## 🛠 Tools Used
- Power BI
- DAX
- Data Modeling
